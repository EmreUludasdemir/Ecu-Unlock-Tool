# Bench Hardware Notes

- Work only with ECUs you own or are explicitly authorized to test.
- Use a current-limited power supply.
- Confirm the correct voltage before connecting hardware.
- Keep wiring notes local to your own bench setup; this project does not provide
  a real pinout database.
- Do not add bypass instructions, boot password recovery steps, or protected
  access procedures.
- Start with read-only experiments.
- Log commands, responses, power events, and test conditions.
- Keep simulator tests passing before and after any hardware experiment.
